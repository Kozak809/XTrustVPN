# XTrustVPN

XTrustVPN is a high-performance VPN client for Android, built on the [sing-box](https://github.com/SagerNet/sing-box) universal proxy platform. It combines a modern Kotlin UI with a powerful Go-based networking core to provide a secure and flexible proxy experience.

## Key Features

- **Powerful Core**: Powered by sing-box, supporting protocols like Shadowsocks, VMess, VLESS, WireGuard, Trojan, and more.
- **Native Performance**: High-speed networking implemented in Go for efficiency and low latency.
- **Modern UI**: A clean, intuitive interface built with Kotlin and Material Design.
- **Flexible Routing**: Advanced routing rules and DNS management.
- **Open Architecture**: Split into a Kotlin-based Android app and a Go-based core (`libcore`).
